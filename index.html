direct in your browser CSS spinner

Using Hakims css spinner as a base we made our own version. It has a smaller element count makes it more useful in a real project, and reduces the file size, and some purely visual tweaks to make it more awesome (in our minds).

Simply:

$ git clone https://github.com/my-codeworks/widgets.git

to get the latest version and drop it into your project. The license is MIT, as usual, and most of the tweaking can be done in the variable section at the top of the SASS file.
